## ▶️ How to Run

1. Clone the repository:

```bash

```

2. Open the project in Visual Studio Code.

3. Make sure JDK 17 (or later) is installed.

4. Open the `src` folder and run any `Main.java` file corresponding to the topic you want to explore.
